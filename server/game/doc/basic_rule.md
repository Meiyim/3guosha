# 三国杀 Online 基础规则

本文档说明当前项目已经实现的规则子集。它面向本地试玩、浏览器开发模式、CLI/机器人测试和早期智能体对局，不等同于完整标准版三国杀规则。

## 游戏目标

游戏采用自由混战生存规则。每名玩家选择一名武将，拥有体力、手牌和装备区。玩家通过出牌和响应使其他角色失去体力，最后存活的玩家获胜。

当前引擎支持 2 名或更多玩家。没有身份牌、主公/忠臣/反贼/内奸阵营，也没有身份局的奖惩和胜利条件。

## 开始游戏

1. 玩家加入同一个房间。
2. 所有座位都加入后进入武将选择。
3. 每名玩家选择一名武将。
4. 游戏发初始手牌并进入第一名玩家的回合。

浏览器普通模式需要玩家输入房间 PIN。开发模式 `?dev=1` 会创建新房间，并启动 AI 客户端加入房间。

## 回合流程

| 阶段 | 当前实现 |
| --- | --- |
| 准备阶段 | 触发已实现的回合开始技能，例如甄姬「洛神」。 |
| 判定阶段 | 阶段存在，但延时锦囊和判定区尚未实现。 |
| 摸牌阶段 | 当前回合玩家从牌堆摸 2 张牌。 |
| 出牌阶段 | 当前回合玩家可以使用合法手牌、发动已实现的主动技能，或结束出牌。 |
| 弃牌阶段 | 若手牌数大于当前体力值，必须弃至等于体力值。 |
| 结束阶段 | 触发回合结束并轮到下一名存活玩家。 |

死亡玩家会从回合顺序和距离计算中跳过。

## 距离与目标

- `杀` 需要选择一名其他存活角色作为目标，不能选择自己。
- 多人局距离按座位环形计算，并忽略死亡角色。
- `+1马` 增加其他角色计算到你的距离。
- `-1马` 减少你计算到其他角色的距离。
- `南蛮入侵` 和 `万箭齐发` 自动以其他所有存活角色为目标。

## 基本牌

| 牌名 | 当前实现 |
| --- | --- |
| 杀 | 出牌阶段对距离内一名角色使用。目标需要出 `闪`，否则受到 1 点伤害。每回合默认限用 1 张。 |
| 闪 | 响应 `杀` 或 `万箭齐发` 时使用，用来抵消对应效果。 |
| 桃 | 出牌阶段为自己回复 1 点体力，不能超过体力上限；濒死救援时可救回濒死角色。 |

## 锦囊牌

| 牌名 | 当前实现 |
| --- | --- |
| 决斗 | 指定一名目标，目标先出 `杀`，双方轮流响应。先不出 `杀` 的角色受到 1 点伤害。 |
| 南蛮入侵 | 除使用者外，每名其他存活角色依次需要出 `杀`，否则受到 1 点伤害。 |
| 万箭齐发 | 除使用者外，每名其他存活角色依次需要出 `闪`，否则受到 1 点伤害。 |
| 无中生有 | 使用者摸 2 张牌。 |

## 装备牌

| 牌名 | 当前实现 |
| --- | --- |
| 诸葛连弩 | 武器。装备后，出牌阶段使用 `杀` 不再受每回合 1 张限制。 |
| +1马 | 防御马。其他角色计算与你的距离时距离 +1。 |
| -1马 | 进攻马。你计算与其他角色的距离时距离 -1。 |

装备牌使用后进入对应装备栏。当前实现不会处理更复杂的装备替换收益或其他武器/防具效果。

## 武将技能

| 武将 | 势力 | 体力 | 当前实现 |
| --- | --- | --- | --- |
| 曹操 | 魏 | 4 | 「奸雄」：受到伤害后，获得造成伤害的实际牌。 |
| 甄姬 | 魏 | 3 | 「洛神」：回合开始时翻牌，黑色牌加入手牌，直到翻到红色牌为止。 |
| 刘备 | 蜀 | 4 | 「仁德」尚未实现。 |
| 关羽 | 蜀 | 4 | 「武圣」：红色非装备牌可以当作 `杀` 使用或响应。 |
| 孙权 | 吴 | 4 | 「制衡」：出牌阶段弃任意张手牌，然后摸等量牌。 |

## 响应、伤害与濒死

- 当规则要求某名玩家响应时，游戏会进入等待状态。
- 被 `杀` 指定时，目标可以出 `闪` 或放弃。
- `决斗`、`南蛮入侵` 和 `万箭齐发` 会按各自规则请求响应。
- 角色体力降至 0 或更低时进入濒死救援流程。
- 救援窗口中可以使用 `桃`。若没有成功救回，濒死角色死亡。
- 当只剩一名存活角色时，该角色获胜。

## 当前未实现的标准规则

- 身份局身份、阵营胜利、击杀奖惩。
- `无懈可击` 和完整锦囊取消窗口。
- 延时锦囊、判定区和完整判定流程。
- 刘备「仁德」。
- 更多标准牌、装备和复杂技能时机。
- 完整标准版三国杀的所有牌堆、角色和扩展规则。

## 操作提示

- 浏览器中点击手牌可以选中或取消选中。
- 需要目标的牌要先选牌，再选择目标，然后点击出牌。
- 等待响应时可以选择可响应的牌，也可以放弃。
- 出牌阶段无合适行动时点击结束出牌。
- 开发模式适合快速测试流程；AI 只是基线策略，不代表强度。
